# Responsive Grid And Spacing Rules v1

## Layout Container
- Max width: 1200px.
- Horizontal padding: 1rem at small, 1.5rem at medium, 2rem at large.

## Grid Rules
- <= 767px: one-column flow.
- 768px to 1023px: two-column content modules where applicable.
- >= 1024px: 12-column grid with span utilities.

## Section Rhythm
- Small screens: 2rem vertical spacing between major sections.
- Medium and larger: 3rem to 4rem based on section density.

## Component Resizing
- Buttons minimum height 44px on all breakpoints.
- Form controls minimum height 44px.
- Trust cards switch from row to stacked layout below 768px.
