# Wireframe Specs v1

## Homepage Wireframe Spec
- Header: logo, primary nav, call CTA, schedule CTA.
- Hero: benefit headline, subhead, primary CTA, secondary CTA.
- Trust strip: badges and review score.
- Services grid: top services with intent-based CTA.
- Financing module: short explanation plus pre-qualify CTA.
- Reviews block.
- FAQ accordion.
- Footer with NAP, hours, compliance links.

## Service Template Wireframe Spec
- Header and breadcrumb.
- Service hero with issue-specific CTA.
- Problem symptoms section.
- Service process timeline.
- Review/trust module.
- Financing CTA block.
- Inline estimate form.
- FAQ and footer.

## Location Template Wireframe Spec
- Header and local proof hero.
- Service availability by neighborhood.
- Local testimonials.
- Click-to-call strip.
- Location-specific FAQ.
- Inline request form.
- Footer.

## Emergency Landing Wireframe Spec
- Minimal header with emergency call CTA.
- Urgency hero with response-time promise.
- One-screen emergency short form.
- Service area validation block.
- Trust and guarantee strip.
- Backup CTA and footer.

## Interaction Notes
- Keyboard-first navigation for all interactive controls.
- CTA order remains consistent across templates.
- Focus ring remains visible at all times on keyboard interaction.
