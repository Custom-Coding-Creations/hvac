# Journey Maps v1

## Journey 1: Emergency User
1. Entry: emergency search or click-to-call ad.
2. Landing: emergency template with immediate call CTA and short emergency form.
3. Decision support: trust bar, response time promise, service area confirmation.
4. Action path A: tap click-to-call.
5. Action path B: submit short emergency request form.

### Friction Controls
- Sticky call bar on mobile.
- One-screen visible emergency form.
- ETA messaging above fold.

## Journey 2: Estimate Shopper
1. Entry: service page from organic search.
2. Landing: service template with benefits, process, social proof.
3. Decision support: pricing ranges, review block, financing teaser.
4. Action path A: request estimate form.
5. Action path B: schedule consultation CTA.

### Friction Controls
- Progressive disclosure FAQ.
- Short form first, optional long details second.
- Persistent request estimate CTA.

## Journey 3: Financing-Constrained User
1. Entry: financing keyword or CTA from service pages.
2. Landing: financing section with qualification summary.
3. Decision support: monthly payment examples and trust guarantees.
4. Action path A: pre-qualify form.
5. Action path B: call financing support.

### Friction Controls
- Clear terms language and plain-English explanations.
- Mobile-friendly collapsible details.
- No mandatory long form before initial interest capture.
