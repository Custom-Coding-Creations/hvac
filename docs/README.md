# Syracuse Website Prospecting Workspace

This folder is a lightweight operating kit for finding and qualifying local website redesign opportunities in Syracuse, NY.

## Files

- `prospect-tracker.csv`: main lead sheet
- `scoring-rubric.md`: ranking model for $2k+ redesign opportunities
- `outreach-templates.md`: reusable email, call, and contact-form templates
- `syracuse-query-pack.md`: search queries and sourcing paths for your first batches

## Recommended First Run

1. Pick one vertical from the query pack.
2. Add 20 businesses to `prospect-tracker.csv`.
3. Disqualify clear non-fits fast.
4. Score the remaining prospects.
5. Personalize outreach for the top 5.
6. Track replies and adjust thresholds.

## Good First Vertical Order

1. HVAC / plumbing / roofing
2. Dentists and specialty clinics
3. Law firms
4. Wedding venues and event businesses
5. Real estate teams

## Minimum Outreach Standard

Only outreach businesses that have:
- an on-site email or contact form
- a visible phone number when possible
- at least one concrete website weakness tied to conversion or trust

## Target Outcome

At the end of the first batch, you should have:
- 20 raw prospects
- 8-12 qualified prospects
- 3-5 outreach-ready businesses
