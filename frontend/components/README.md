# Frontend Components

This directory describes reusable component groups used by the template files.

## Component Groups
- Header and navigation system with mobile disclosure behavior.
- Trust modules including badges, reviews, and financing confidence blocks.
- CTA button system with primary and secondary variants.
- Form patterns with accessible labels and inline errors.
- FAQ accordion and semantic footer.

## Extension Rules
- Reuse classes from frontend/assets/css/system.css before creating new ones.
- If a new token is needed, add it in design-system/TOKENS.md first.
- New interactive components must provide keyboard support and visible focus styles.
